# SwipeBackLayoutTouTiao
仿今日头条、腾讯新闻界面滑动关闭 下层activity有缩放效果
特点：  
  1.仿今日头条滑动返回效果，底层anctivity有缩放效果    
  2.解决了透明背景activity无法进入onStop问题,打开多个界面也不会出现卡顿问题
  3.支持边缘滑动、全屏滑动
  4.支持和沉浸式状态栏一起滑动返回
  5.支持viewpager、listview、webview等，处理了滑动冲突
  6.解决了华为手机虚拟键遮挡布局问题
